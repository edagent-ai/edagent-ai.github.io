# EdAgent.AI

> Open-source AI tools for education

We build free, open-source AI-powered tools that make learning more accessible, personalized, and goal-driven — for students, self-learners, and educators everywhere.

## What we build

- **Adaptive tutoring** — AI tutors that adjust to each learner's pace, level, and style in real time
- **Self-learning curriculum design** — Tools that help learners build their own structured curriculums around any topic
- **Learning goal planning with AI** — Plan, track, and achieve learning goals with AI-guided milestones and progress insights

## Principles

- **Open by default** — Every line of code is publicly available
- **Student privacy** — Data is never sold or shared
- **Learner-led** — Tools designed to put the learner in control
- **Equity-focused** — Free to use, built for everyone
