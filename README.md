# EdAgent.AI

> Open-source AI tools for education

We build free, open-source AI-powered tools that make education more accessible, adaptive, and effective — for students, teachers, and self-learners everywhere.

## What we build

- **Adaptive tutoring** — AI tutors that meet students at their level
- **Smart feedback** — Instant, detailed feedback on essays, code, and problem sets
- **Curriculum mapping** — Tools for educators to design effective learning paths
- **Learning analytics** — Transparent dashboards that surface where students struggle
- **Multilingual support** — Breaking language barriers in education
- **Accessibility first** — Built for diverse learning needs and low-bandwidth environments

## Principles

- **Open by default** — Every line of code is publicly available
- **Student privacy** — Data is never sold or shared
- **Educator-led** — Teachers and learners shape every tool
- **Equity-focused** — Free to use, designed for under-resourced schools first
